# $cpu
Возвращает ЦПУ нагрузку сервера
### Использование
```php
$cpu[метод?]
```

### Опции

| Опция | Описание | Тип | Обязательно |
|--------|-------------|------|----------|
| метод | метод для ЦПУ | текст | Нет |  
## Пример(ы)

```javascript
bot.command({
  name: '$cpu',
  code: `
$cpu`
// Возвращает: 0.081
})
```
