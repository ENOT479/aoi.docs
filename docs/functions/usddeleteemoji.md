# $deleteEmoji
Удаляет указанный эмоджи
### Использование
```php
$deleteEmoji[эмоджи]
```

### Опции

| Опция | Описание | Тип | Обязательно |
|--------|-------------|------|----------|
| эмоджи | эмоджи, который нужно удалить | айди | Да |  
## Пример(ы)

```javascript
bot.command({
  name: '$deleteEmoji',
  code: `
$deleteEmoji[$message[1]]`
})
```
