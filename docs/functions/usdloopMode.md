# $loopMode
Ставит трек или очередь на повтор
### Использование
```php
$loopMode[режим?]
```

### Опции

| Опция | Описание | Тип | Обязательно |
|--------|-------------|------|----------|
| режим | Что будет зациклено | Строка | Нет |  

## Режимы
  queue - зациклить очередь
  song - зациклить трек
  none - отключить повтор

## Пример(ы)

```javascript
bot.command({
  name: 'loopMode',
  code: `
$loopMode[queue]`
})
```
