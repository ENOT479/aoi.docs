# $emojiExists
Возвращает логическое выражение если указанное эмоджи существует
### Использование
```php
$emojiExists[эмоджи]
```

### Опции

| Опция | Описание | Тип | Обязательно |
|--------|-------------|------|----------|
| эмоджи | эмоджи, которое нужно проверить | текст | Да |  
## Пример(ы)

```javascript
bot.command({
  name: '$emojiExists',
  code: `
$emojiExists[dQw4w9WgXcQ]`
// Возвращает: true
})
```
