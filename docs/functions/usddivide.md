# $divide
Делит указанные числа
### Использование
```php
$divide[...числа]
```

### Опции

| Опция | Описание | Тип | Обязательно |
|--------|-------------|------|----------|
| ...числа | числа для деления | перечисление | Да |  
## Пример(ы)

```javascript
bot.command({
  name: '$divide',
  code: `
$divide[1000;10;10;10]`
// Возвращает: 1
})
```
