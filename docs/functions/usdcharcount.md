# $charCount
Возвращает количество символом в указанном тексте
### Использование
```php
$charCount[текст?]
```

### Опции

| Опция | Описание | Тип | Обязательно |
|--------|-------------|------|----------|
| текст | текст, символы которого нужно подсчитать | текст | Нет |  
## Пример(ы)

```javascript
bot.command({
  name: '$charCount',
  code: `
$charCount[текст?]`
// Возвращает: 6
})
```
