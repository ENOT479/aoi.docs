# $deleteWebhook
Удаляет вебхук
### Использование
```php
$deleteWebhook[айди;токен]
```

### Опции

| Опция | Описание | Тип | Обязательно |
|--------|-------------|------|----------|
| айди | айди вебхука | айди | Да | 
| токен | токен вебхука | текст | Да | 
## Пример(ы)

```javascript
bot.command({
  name: '$deleteWebhook',
  code: `
$deleteWebhook[$message[1];$message[2]]`
})
```
