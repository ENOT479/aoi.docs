# $deleteFile
Удаляет указанный файл на хостинге бота
### Использование
```php
$deleteFile[файл]
```

### Опции

| Опция | Описание | Тип | Обязательно |
|--------|-------------|------|----------|
| файл | файл, который нужно удалить | путь | Да |  
## Пример(ы)

```javascript
bot.command({
  name: '$deleteFile',
  code: `
$deleteFile[index.js]`
})
```
