# $digitalFormat
Переводит время в цифровой формат
### Использование
```php
$digitalFormat[миллисекунды]
```

### Опции

| Опция | Описание | Тип | Обязательно |
|--------|-------------|------|----------|
| миллисекунды | миллисекунды для перевода в цифровой формат | число | Да |  
## Пример(ы)

```javascript
bot.command({
  name: '$digitalFormat',
  code: `
$digitalFormat[$parseTime[10h]]`
// Возвращает: 10:00:00
})
```
