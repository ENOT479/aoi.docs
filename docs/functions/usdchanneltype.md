# $channelType
Возвращает тип канала
### Использование
```php
$channelType[канал?]
```

### Опции

| Опция | Описание | Тип | Обязательно |
|--------|-------------|------|----------|
| канал | канал, тип которого нужно узнать | айди | Нет |  
## Пример(ы)

```javascript
bot.command({
  name: '$channelType',
  code: `
$channelType[987109762073662018]`
// Возвращает: Text
})
```
