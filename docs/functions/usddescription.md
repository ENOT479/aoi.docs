# $description
Создаёт описание в встроенном сообщении (эмбед)
### Использование
```php
$description[индекс?;текст]
```

### Опции

| Опция | Описание | Тип | Обязательно |
|--------|-------------|------|----------|
| индекс | индекс встроенного сообщения | число | Да |  
| текст | текст для описания | текст | Да |  

## Пример(ы)

```javascript
bot.command({
  name: '$description',
  code: `
$description[1;$message]`
})
```
