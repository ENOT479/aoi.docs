# $color
Устанавливает цвет для встроенного сообщения. Цвет нужно указывать в HEX формате.
### Использование
```php
$color[индекс;цвет]
```

### Опции

| Опция | Описание | Тип | Обязательно |
|--------|-------------|------|----------|
| индекс | индекс встроенного сообщения | число | Да |  
| цвет | hex цвет | текст | Да |
## Пример(ы)

```javascript
bot.command({
  name: '$color',
  code: `$title[1;Hello World!]
$color[1;000001]`
})
```
